## 1. Que es Pedily Realmente

Pedily NO es solo "una web para vender". Es un **ecosistema completo multicanal** que incluye:

| Componente | Descripcion | Valor Individual |
|------------|-------------|------------------|
| **Web Storefront** | Tienda online completa con catalogo, carrito, checkout, cuentas de usuario | $99-199/mes |
| **WhatsApp Bot con IA** | Bot conversacional con Claude AI que toma pedidos 24/7 | $45-129/mes |
| **Kiosko Autoservicio** | App para tablet con integracion POS (IZIPAY) | $50-99/mes + hardware |
| **Hub Administrativo** | Dashboard de operaciones, gestion de pedidos, roles, reportes | $50-100/mes |
| **Facturacion SUNAT** | Boleta/Factura electronica automatica via Nubefact | S/ 40-100/mes |
| **CRM + Fidelizacion** | Base de clientes, puntos, cupones, referidos | $30-50/mes |
| **Integracion Delivery** | PedidosYa Envios para logistica | Variable |

**Costo total si un restaurante armara esto por separado: $324-677/mes + setup**

**Pedily Empresa lo da todo por: S/ 999/mes (~$264 USD)**

## 5. Problema: El Pricing Esta SUBVALORADO

### Evidencia de Subvaloracion

1. **Comparacion internacional:**
   - ChowNow (solo web): $119-298/mes
   - Toast + Kiosk: $138+/mes
   - Pedily (3 canales): $47-185/mes

2. **Costo de desarrollo:**
   - Construir esta plataforma costo cientos de miles de dolares
   - El pricing no refleja el valor entregado

3. **Percepcion de mercado:**
   - Precio muy bajo puede percibirse como "producto inferior"
   - Restaurantes establecidos esperan pagar mas por soluciones enterprise

4. **Margen de negociacion:**
   - Sin espacio para descuentos corporativos
   - Sin margen para partnerships o resellers

### Benchmark de Valor por Feature

| Feature | Valor de Mercado | En Pedily |
|---------|------------------|-----------|
| Web Store con checkout | $100-200/mes | Incluido desde Inicio |
| WhatsApp Bot con IA | $50-150/mes | Incluido desde Crecimiento |
| Kiosko + POS | $100-200/mes | Incluido en Empresa |
| Facturacion SUNAT | S/ 40-100/mes | Incluido desde Crecimiento |
| CRM + Loyalty | $30-80/mes | Incluido desde Crecimiento |
| Hub Multicanal | $50-100/mes | Incluido en todos |

## 6. Pricing Premium Implementado

Se implemento la estrategia de pricing premium para reflejar el valor real del producto.

### Pricing Final (Febrero 2026)

| Plan | Precio Mensual | Precio Anual | USD Equiv. |
|------|----------------|--------------|------------|
| **Inicio** | S/ 249 | S/ 199/mes | ~$66 |
| **Crecimiento** | S/ 499 | S/ 399/mes | ~$132 |
| **Empresa** | S/ 999 | S/ 799/mes | ~$264 |

### Justificacion del Pricing Premium

- Sigue siendo 43% mas barato que armar por separado (~$450/mes)
- ROI vs agregadores sigue siendo masivo (98%+ ahorro)
- Posiciona como solucion profesional, no "barata"
- Margen para descuentos corporativos (15-20%)
- Espacio para futuro plan Enterprise custom

### Futuro: Plan Enterprise (S/ 1,499+/mes)

Para cadenas y franquicias. Incluiria:
- Soporte 24/7 dedicado
- SLA garantizado 99.99%
- Integraciones custom (ERP, contabilidad)
- Multi-RUC / Multi-marca
- Account manager dedicado
- Capacitacion presencial
- White-label completo

## 10. Conclusiones y Recomendaciones

### Lo que Pedily hace bien:
1. **Producto completo** - Ningun competidor ofrece los 3 canales
2. **Tech stack moderno** - Next.js, NestJS, Supabase, Claude AI
3. **Integraciones locales** - SUNAT, Culqi, IZIPAY, Yape/Plin
4. **UX cuidada** - Interfaces pensadas para usuarios no tecnicos

### Lo que debe mejorar:

1. **Pricing:** Subvalorado para el mercado. Subir 40-50%.

2. **Posicionamiento:** No es "una tienda web". Es "plataforma multicanal con IA". El mensaje debe reflejarlo.

3. **Tier Enterprise:** Falta plan para cadenas (S/ 1,500+/mes).

4. **Metricas de valor:** Mostrar ROI calculado vs agregadores en el sitio.

5. **Case studies:** Publicar resultados reales (Wantan Clan +40% pedidos).

### Accion Inmediata Recomendada:

1. **Subir precios** a S/ 249 / S/ 499 / S/ 999
2. **Agregar calculadora de ROI** en landing ("Cuanto ahorras vs Rappi")
3. **Crear tier Enterprise** con pricing custom
4. **Redactar caso de exito** detallado de Wantan Clan
